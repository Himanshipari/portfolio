## 📖 About The Project

This portfolio serves as my digital presence, highlighting my technical expertise, development projects, certifications, and career aspirations. The website is designed with a focus on performance, responsiveness, and user experience.

It provides visitors, recruiters, and hiring managers with a comprehensive overview of my skills, projects, and contact information in a professional and visually appealing format.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Himanshipari/portfolio.git
```

### Navigate to Project

```bash
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
